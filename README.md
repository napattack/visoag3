# visoag3
this is group 3 analysis repository of visualisation open access 

This repository contains two folders data and plots. 
**data** contains the original files
**plots** contains all plots results 

we has three r files. 
***score.R*** to transform our original data, and give score to each institution
***api_wikidata.R*** to query the institution location 
***mapping.R*** to map the geolocation into a map
